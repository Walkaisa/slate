    Introduction

    Welcome to the Walkaisa API! This API is designed to improve your project by providing a wide range of endpoints for various functionalities. With Walkaisa API, you can easily integrate many features into your application, including discord channel transcripts, OAuth2 verification, member restore, and more.

    Authentication

    To authorize, use this code:

    import Axios from "axios";

const result = await Axios.get("https://api.walkaisa.dev", {
    headers: {
        Authorization: "YOUR_KEY"
    }
});

    Make sure to replace YOUR_KEY with your API key.

    The API uses keys to allow access to the API. You can receive a new API key by joining our Discord.

    The API expects for the API key to be included in all API requests to the server in a header that looks like the following:

    Authorization: YOUR_KEY

    You must replace YOUR_KEY with your personal API key.

    Transcript

    Create transcript

    import Axios from "axios";

const result = await Axios.post(
    "https://api.walkaisa.dev/transcripts",
    {
        token: "MTA0ODMzMDc1Mzg2NTQzMzEwOQ.G3tbI5...",
        guild_id: "893502016024875008",
        channel_id: "1089202275383255041",
        header: {
            Guild: "Walkaisa Industries",
            Channel: "general"
        },
        footer: ["Powered by Walkaisa Industries"]
    },
    {
        headers: {
            Authorization: "YOUR_KEY"
        }
    }
);

    The above request returns JSON structured like this:

    {
    "statusCode": 200,
    "response": "BufferBase64"
}

    This endpoint returns a base64 encoded buffer of the generated html transcript.

    HTTP Request

    GET https://api.walkaisa.dev/transcripts

    Body Parameters

    Parameter Description
    token The token of the Discord bot.
    guild_id The ID of the guild.
    channel_id The ID of the channel.
    header The header of the transcript. (A custom key value object.)
    footer The footer of the transcript. (A array of strings.)
    200 - OK

    Crypto

    Get all available Cryptocurrencies

    import Axios from "axios";

const result = await Axios.get("https://api.walkaisa.dev/crypto", {
    headers: {
        Authorization: "YOUR_KEY"
    }
});

    The above request returns JSON structured like this:

    {
    "statusCode": 200,
    "response": [
        {
            "id": "bitcoin",
            "name": "Bitcoin",
            "symbol": "btc"
        },
        {
            "id": "ethereum",
            "name": "Ethereum",
            "symbol": "eth"
        }
    ]
}

    This endpoint retrieves all available cryptocurrencies.

    HTTP Request

    GET https://api.walkaisa.dev/crypto

    200 - OK

    Get a specific Cryptocurrency

    import Axios from "axios";

const result = await Axios.get("https://api.walkaisa.dev/crypto", {
    headers: {
        Authorization: "YOUR_KEY"
    },
    params: {
        query: "bitcoin"
    }
});

    Errors

    The Walkaisa API uses the following error codes:

    Error Code Description
    400 Bad Request -- Your request is invalid.
    401 Unauthorized -- Your API key is wrong.
    403 Forbidden -- You do not have permission to access the requested resource.
    404 Not Found -- The specified resource could not be found.
    429 Too Many Requests -- You're making too many requests! Please try again later.
    500 Internal Server Error -- We had a problem with our server. Try again later.
    503 Service Unavailable -- We're temporarily offline for maintenance. Please try again later.
